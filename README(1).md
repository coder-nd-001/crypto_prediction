# CryptoSage AI 🚀

## Overview

CryptoSage AI is an intelligent cryptocurrency price prediction platform
built using Machine Learning and Deep Learning models. It predicts
future prices of cryptocurrencies like Bitcoin, Ethereum, Litecoin,
Solana, Cardano, and Dogecoin.

## Features

-   Live crypto market dashboard
-   Price prediction using:
    -   LSTM
    -   Linear Regression
    -   Hybrid Model
-   Technical Indicators:
    -   RSI
    -   MACD
    -   Bollinger Bands
-   Risk analysis
-   Portfolio simulation
-   Manual model training with uploaded CSV

## Tech Stack

-   Python
-   Streamlit
-   TensorFlow
-   Scikit-learn
-   Pandas
-   NumPy
-   Plotly
-   CoinGecko API

## Project Workflow

1.  Fetch market data
2.  Preprocess data
3.  Generate technical indicators
4.  Train ML/DL models
5.  Predict next price
6.  Visualize insights

## Models Used

### LSTM

Captures sequential dependencies in price data.

### Linear Regression

Fast baseline prediction.

### Hybrid Model

Combines LSTM + Linear Regression for better accuracy.

## Installation

``` bash
pip install -r requirements.txt
streamlit run app.py
```
